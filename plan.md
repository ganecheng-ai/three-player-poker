# 斗地主游戏项目规划

## 项目概述
使用 Python 开发一个经典的三人斗地主游戏，支持简体中文界面，画面精美，操作流畅。

## 技术选型
- **编程语言**: Python 3.10+
- **图形界面**: Pygame
- **打包工具**: PyInstaller
- **本地化**: gettext 或自定义 JSON 翻译文件

## 游戏功能

### 核心功能
1. **扑克牌系统**
   - 54 张牌（含大小王）
   - 牌面渲染和动画效果

2. **发牌系统**
   - 随机洗牌
   - 三人各 17 张，底牌 3 张

3. **叫地主系统**
   - 轮流叫地主
   - 叫分逻辑（1 分、2 分、3 分、不叫）

4. **出牌系统**
   - 支持所有斗地主牌型
   - 单张、对子、三张、三带一、三带二、顺子、连对、飞机、炸弹、王炸
   - 牌型比较和验证

5. **AI 系统**
   - 智能出牌决策
   - 记牌器功能

6. **界面系统**
   - 精美卡牌设计
   - 流畅动画效果
   - 简体中文界面

### 牌型规则
| 牌型 | 说明 | 示例 |
|------|------|------|
| 单张 | 任意一张牌 | 3, 4, 5...2, A, K, Q, J, 10, 9, 8, 7, 6, 5, 4, 3 |
| 对子 | 两张点数相同的牌 | 33, 44, 55... |
| 三张 | 三张点数相同的牌 | 333, 444, 555... |
| 三带一 | 三张相同 + 一张单牌 | 333+4 |
| 三带二 | 三张相同 + 一对牌 | 333+44 |
| 单顺子 | 五张或以上连续单牌 | 34567, 45678... |
| 双顺子 | 三个或以上连续对子 | 334455, 445566... |
| 飞机 | 两个或以上连续三张 | 333444, 444555... |
| 飞机带翅膀 | 飞机 + 相同数量的单牌或对子 | 333444+56, 333444+5566 |
| 炸弹 | 四张相同点数 | 3333, 4444... |
| 王炸 | 大王 + 小王 | 最大牌型 |
